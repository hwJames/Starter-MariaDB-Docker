Docker-MariaDB
</h2>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hwJames/Docker-MariaDB.git
   git pull
   ```

2. Make Directory
   ```sh
   mkdir data
   mkdir initdb.d
   ```

3. Edit Setting
   ```sh
   vi .env
   vi docker-compose.yml
   ```

4. Docker Compose Start
   ```sh
   docker-compose up
   ```
